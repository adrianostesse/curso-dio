Tags H1 a H6

Representam niveis de titulos, basicamente a letra vai ser maior no H1 e ir diminuindo até o H6. Meio que mostrando o "nivel de importância" (mesmo sendo todos importantes)

Tags de Formatação de texto

Mark - Faz uma marcação como se fosse um marca-texto, dessa forma

Small - Faz o texto ficar menor, dessa forma

i - Faz o texto ficar em itálico, dessa forma

u - Faz o texto ficar sublinhado, dessa forma

Strong - Faz o texto ficar em negrito,dessa forma

sub - Faz o texto ficar subescrito,dessa forma (mais em baixo)

sup - Faz o texto ficar sobrescrito,dessa forma (mais em cima)

Listas ordenadas e não ordenadas

Usa-se as Tags OL e UL para fazer listas. OL para listas ordenadas e UL para não ordenadas. E usa o LI dentro para marcar a lista de itens, sendo de forma ordenada ou não. EX:

  1. lista ordenada fica assim
  2. por padrão ele fica com numeros de ordem
  • Lista não ordenada fica assim
  • Por padrão com uma bolinha do lado

Tag Blockquote

Para fazer citações dentro do site

o texto citado fica desse jeito, mais deslocado. Bom para citar autores e/ou frases deles entre outras citações.

Tag Abbr

Define uma abreviação e pode dar uma descrição dela. Ex: <abbr tittle="Hypertext Markup Language">HTML</abbr>
